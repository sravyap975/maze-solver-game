# maze-solver-game
Key Features:
Procedural Generation: Uses Recursive Backtracking to carve out a unique, "perfect" maze for every level, ensuring the exit is always reachable.

AI Pathfinding: Features a Breadth-First Search (BFS) hint system that calculates and highlights the shortest path from the player to the goal.

Progressive Difficulty: The grid expands and the time limit scales as players level up, increasing the challenge.

Modern UI: A high-contrast "Neon" aesthetic built with CSS Grid and glowing visual effects.

Tech Stack:
HTML5/CSS3: Grid layouts and neon styling.

JavaScript: Core game engine and algorithmic logic.
